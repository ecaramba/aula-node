# Sistema de agendamento consultório

## Requisitos

- Agendamentos do Dia
- Agendamento da Consulta
- Cadastro de Clientes
- Cadastro de Serviços (preço)
- login
- Cadastro de usuário
- Cadastro Profissionais

## Requisitos tecnicos
- acessivel via celular
- rodar no desktop
- internet / nuvem

## implementação
- frontend -> bootstrap + jquery
- backend -> api node + mongodb
